# binary wheel, and adding it to the production install risks the nixpacks
# Docker apt/pip layer-cache guard rails (slow deploys) AND changes prod
# behavior. It lives here so CI and dev environments can exercise the
# fitz-gated tests (tests/spine/test_flatten.py, test_visual_qa.py,
# test_routes_inspector.py), which skipif PyMuPDF is absent.
#
# Install:  pip install -r requirements-dev.txt
